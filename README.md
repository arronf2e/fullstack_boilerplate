# pnpm monorepo boilerplate

## Preconditions

- android sdk
- xcode
- nodejs
- pnpm

## 1. Framework

### 1.1. frontend

- react
- vite
- typescript

### 1.2. app

- react native
- expo

### 1.3. backend

- nestjs

## 2. Start

```shell
#start web
- pnpm -w run start:web

#start ios
- pnpm -w run start:ios

#start android
- pnpm -w run start:android

# start server
- pnpm -w run start:server
```
